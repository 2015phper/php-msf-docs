## 运行

### build.sh(Docker环境)

```bash
$>cd php-msf-demo
$>./build.sh
```

### 登入Docker容器
```
$> ssh -p 2202 worker@127.0.0.1
密码："worker"(无双引号)
```

其中`2202`端口是根据映射`-p 2202:22`来的，如果有修改请调整为修改后的值。

### composer

```bash
$>cd /home/worker/data/www/php-msf-demo
$>composer install -vvv
```
